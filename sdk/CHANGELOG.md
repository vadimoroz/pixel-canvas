# Changelog

## [1.0.0] - 2025-03-06

### Added
- Initial release
- `PixelCanvasClient` class with read-only methods
- `getPixel(x, y)` — fetch pixel data at coordinates
- `getPixelCount(address)` — get painter's pixel count
- `getTotalPixels()` — get total pixels placed
- `getCanvasSize()` — get canvas dimensions
- TypeScript types: `Pixel`, `CanvasSize`, `PlacePixelOptions`
- Contract constants exported for frontend use
